# Camotec (Камотек) — Профіль компанії

> **Короткий висновок (5 рядків):** Camotec — одеський повноциклний виробник функціонального тактичного одягу та спорядження, заснований у 2013 р. Засновник і власник — Сергій Шевельов. Компанія сертифікована як виробник («повний цикл — від ідеї до продажу»), має 81–200 співробітників, активно розширює роздріб (Київ-флагман грудень 2024). Юридична і торгова адреса в Одесі — вул. Вернадського, 54 — це роздрібний магазин та офіс, а не окремий виробничий цех. Наш об'єкт на Дальницькій може закрити потребу у власному виробничому майданчику, якого компанія явно не має у публічному просторі: швейний цех 500–1500 м² + шоурум + склад + охоронювана територія для відвантаження / прийому матеріалів.

## 4. Формат нерухомості та вимоги

Специфічних публічних вимог до нерухомості Camotec не публікує. Оцінка на основі профілю швейного виробника 81–200 осіб:

| Параметр | Типова потреба (оцінка) | Наш об'єкт | Відповідність |
|---|---|---|---|
| Площа цеху | 500–1500 м² (цех + склад матеріалів + склад ГП + офіс) | 1500 м² total | Точне попадання |
| Поверховість | 1–2 поверхи | 2 поверхи | Підходить (2 сходи = поділ цех/офіс) |
| Висота стель | 2,8–4 м (швейні машини, розкрійні столи, манекени) | 2,8–3,0 м | Прийнятно (нема потреби у кранових балках) |
| Електромережа | 30–60 кВт (промислові швейні машини, розкрійні комплекси, прасувальне обладнання, вентиляція) | 380В / 50 кВт | Підходить для цеху 50–80 машин |
| Вентиляція | Приточно-витяжна (пил від тканин, хімія від фарбування) | Монтаж можливий (є виводи) | Потребує монтажу орендарем |
| Водопостачання | Централізоване (прасування, прибирання) | Центральний водопровід | Підходить |
| Охорона і відеонагляд | Обов'язкова (тактичний одяг = цінний товар, військова специфіка) | Пожежна + охоронна («Легіон») + відеоспостереження + паркан + ворота | Повна відповідність |
| Двір і логістика | Для прийому рулонів тканини, відправки ГП ЗСУ і в роздріб | 1028 м² + ворота 5 м (фура до 10 м) | Підходить |
| Шоурум-складова | 200–400 м² для демонстрації і B2B-клієнтів | Фасад 28 м + витринне скління | Ідеально для шоуруму |
| Парковка | 5–10 місць (персонал, кур'єри, B2B-клієнти) | 6–7 місць + вулична | Достатньо |
| Тип локації | Стрит (промзона) | Стрит, Хаджибейська промзона | Природна відповідність |

**Принципова перевага об'єкта:** поєднання виробничого цеху (тил будівлі) + шоуруму (фасад із 28 м витрини) + закритого двору для логістики — в одному просторі, без оренди трьох окремих об'єктів.

**Ключове ускладнення:** 50 кВт — достатньо для цеху з 60–80 промисловими швейними машинами. Якщо Camotec планує автоматизовані розкрійні комплекси (GERBER, LECTRA) — може знадобитись 60–80 кВт. Рекомендується власнику заздалегідь з'ясувати можливість збільшення ліміту через Одесаобленерго.

## 6. «Гачки» — персональні аргументи для Camotec

Не загальні плюси об'єкта, а конкретні попадання у ситуацію компанії:

**1. «Ви наймаєте швачок і конструкторів — але де ваш цех?»**
Активні вакансії 2026 (швачка + конструктор одягу) в Одесі без публічного виробничого майданчика = або цех перевантажений, або шукають нові площі. Наш об'єкт — готова відповідь: 1500 м², кліматизація під монтаж, 380В/50кВт.

**2. Вернадського, 54 — це не виробничий цех, а торговий простір**
Юрадреса і магазин в центрально-комерційній частині Одеси (Київський район) не підходить для розкрійного цеху. Промзона Хаджибейського — саме туди переїжджають виробничі функції одеських компаній. Дальницька = природна виробнича вісь міста.

**3. Шоурум-флагман «для захисників» прямо у промзоні — новий формат**
Відкривши Kyiv-флагман, Шевельов показав: компанія будує брендові торгові точки. Наш об'єкт — можливість відкрити одеський виробничий флагман: «Вхід — шоурум, за ним — цех». Для B2B-клієнтів (держзамовлення, оптові покупці) — живий доказ «виробник, а не перекупник».

**4. 28 м фасаду на автомобільній артерії = безкоштовна реклама**
Дальницька — головна виїзна вісь Одеси (стабільний денний трафік). Вивіска «Виробник тактичного одягу Camotec» зі зручним заїздом — видно всім, хто їде через Хаджибейський район. Клієнти-військові і оптовики їдуть цілеспрямовано — автомобільний трафік = ваша аудиторія.

**5. Огороджена охоронювана територія — базова вимога для тактичного виробника**
Пожежна, охоронна сигналізація, відеоспостереження, паркан по периметру, закриті ворота — вже є. Для виробника тактичного і військового спорядження (B2B-клієнти: ЗСУ, НГУ, ДПСУ) — це не опція, а вимога безпеки. Монтажу не потрібно.

**6. Двір 1028 м² — логістика для прийому тканин і відправки ГП**
Прийом рулонів тканини (прибуває 5-тонниками), відправка готової партії у ЗСУ або в роздрібні точки — всередині двору, без паркування фур на Дальницькій і без залежності від міської парковки.

**7. Гнучке зонування без капітальних робіт**
Перегородки — гіпсокартон, демонтуються. Перший поверх: розкрій + швейний цех. Другий поверх: конструкторське бюро + офіс + склад готової продукції + QC-зона. Або: фасадна частина — шоурум B2B-клієнтів, тил + двір — виробництво і логістика. Все без втручання в несучі конструкції.
