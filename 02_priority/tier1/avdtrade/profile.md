## 4. Формат нерухомості

У AVDtrade немає формату «роздрібний магазин для кінцевих покупців» у класичному розумінні. Компанія оперує **регіональними складами-відділеннями** для B2B-обслуговування партнерів (СТО, авторитейлери).

| Параметр | Вимога / типовий формат |
|---|---|
| Тип об'єкта | Склад + відділення (НЕ роздрібний магазин, НЕ ТРЦ) |
| Площа | Не підтверджено публічно; аналог — центральний склад Житомир 5 000 м²; регіональні — ймовірно 500–1500 м² |
| Парковка | Потрібна для вантажного транспорту (доставка товару і відпуск клієнтам-СТО) |
| Електромощність | Не підтверджено публічно; для складу + офісу достатньо 50 кВт |
| Висота стелі | Не підтверджено; типово для авто-складу — 4–6 м; 2,8–3,0 м є обмеженням для стелажного зберігання |
| Фасад / вивіска | Не критично — B2B-клієнти знають куди їхати |
| Вантажний в'їзд | Критично: потрібні ворота для вантажівок (у нас 5 м — відповідає) |
| Місцевість | Промзона або промавтомагістраль — ідеально для маршрутної логістики |

**Обмеження об'єкта для AVDtrade:**
Висота стелі нашого об'єкта — 2,8–3,0 м (у складській ангарній частині). Для повноцінного стелажного авто-складу (паллети + автокар) потрібно мінімум 4–5 м. Тобто об'єкт підходить як **шоу-склад + регіональний офіс відпуску**, але НЕ як класичний логістичний авто-хаб із висотними стелажами. Цей момент потрібно проговорювати чесно.

## 6. «Крючки» — що зачепить AVDtrade конкретно

1. **Одеса — єдиний великий регіон без власного відділення.** Компанія має регіональні склади в Житомирі, Києві, Львові та Вінниці. Одеси в цій карті немає. Дальницька/Хаджибейський район — ідеальна «промислова точка входу» в порт і південь: через трасу йде трафік у Миколаїв, Херсон, Молдову.

2. **Двір 1 028 м² + ворота 5 м = логістична інфраструктура без доплати.** Для B2B-складу автозапчастин це критично: розвантаження фур + місце для клієнтів з СТО, які приїжджають на своїх мікроавтобусах або малотоннажниках. У нас є двір, куди заходить вантажівка до 10 м, і ворота без обмежень по висоті.

3. **Сусідство з АЗС = постійний цільовий трафік.** Механіки, СТО, власники авто, що в'їжджають на заправку, — це і є потенційні клієнти AVDtrade. Вивіска + шоу-склад в 500 м від АЗС на виїзді з Одеси можуть дати органічних walk-in клієнтів зверх оптової бази.

4. **380 В / 50 кВт — без апгрейду інфраструктури.** Для складу + відділу продажів + офісу + навантажувально-розвантажувального обладнання потужність 50 кВт є достатньою.

5. **Об'єкт підходить під формат «шоу-склад + регіональний офіс».** Фасадна частина — вітринний показ топових брендів (Brembo, KYB, Hella, GATES тощо). Задня ангарна частина — склад на паллетах + зона відпуску. Дві незалежні зони на 2-му поверсі — окремо офіс для регіонального менеджера + кімнати для навчання партнерів (AVDtrade активно розвиває корпоративне навчання через moodle.avdtrade.com.ua).

6. **Ставка $2/м² = вдвічі нижча ринкової для промислово-складської нерухомості Одеси ($3–5/м²).** Для регіонального відділення із виручкою від 500+ оптових клієнтів юга — це витрати, що окупаються за 1–2 місяці додаткового обороту.

7. **«Промзона, куди їдуть цілеспрямовано».** AVDtrade не потребує пішохідного трафіку — їхні клієнти (СТО та авторитейлери) ЗАВЖДИ їдуть на машині та заздалегідь знають адресу. Хаджибейський район — природна локація для такого B2B-формату (42% промпідприємств міста, маршрутна логістика).
