Публічно не декларуються. На основі профілю операцій:

- Типовий склад FMCG-дистрибьютора: висота стель **6–10 м** (паллетне зберігання в 3–5 ярусів із навантажувачами). **Наш об'єкт (стелі 2,8–3,0 м) не відповідає цьому стандарту.**

## 8. «Гачки» для комерційної пропозиції

1. **Одесса — білий регіон на карті GD.** Публічне покриття Grand Distribution не включає Одесу. Якщо це не навмисне рішення, а незакрита зона — сателітний склад 1500 м² на Дальницькій (головна транспортна артерія на Київ / Миколаїв / Херсон) закриє логістичне вікно одразу для Одесько-Миколаївського кластера.
2. **Ціна нижче ринку в 2–2,5 рази.** $2/м² проти ринкових $4–5/м² для складу B в Одесі — значна операційна економія на регіональному рівні.
3. **Цементний підлог + двір 1000 м².** Прийом рефрижераторів та газелей без виїзду на вулицю, безпечна розвантаження за будь-якої погоди.
4. **Охорона та відеоспостереження вже змонтовані.** Для FMCG (алкоголь, косметика, кондитерські) — важлива обставина без додаткових вкладень.
5. **50 кВт / 380В.** Достатньо для холодильних і морозильних камер у зоні до 200–300 м² (за умови уточнення у поточного власника).

## 9. Критичні обмеження (чесний фрейм)

1. **Стелі 2,8–3,0 м** — критичне обмеження для операційного FMCG-складу із навантажувачами. Потрібно прямо проговорювати в першому листі: «формат — 1–2-ярусний паллетний склад рохлями, не висотний».
2. **Один в'їзд у двір.** Grand Distribution на дніпровському HQ обробляє 490 т/день (25–50 фур). Об'єкт фізично може прийняти 3–5 фур/день — прийнятно для сателіту, але не для основного хаба.
3. **Відсутність доків/рамп.** FMCG-дистрибьютори полюбляють рівні рампи для навантажувачів. На об'єкті — розвантаження з борту вантажівки. Для газелей прийнятно; для великотоннажних машин — операційна незручність.
4. **Температурні режими під питанням.** Для Carlsberg, Lactalis, Nemiroff потрібні зони +4…+8°C та +18°C. Об'єкт зараз без газового/промислового холодильного обладнання. Арендатор сам встановлює.
