# Velmet (ТОВ «Велмет») — Профіль компанії

> Summary: Velmet — хмельницький виробник засобів захисту, тактичного одягу та спорядження. Заснований у 2011 р. як волонтерський рух, зараз виконує держконтракти МВС і ЗСУ (сумарно ≥280 млн грн з 2017 р., контракт МВС 53 млн грн у листопаді 2025). Штат: 51–200 осіб (LinkedIn). Власник і директор — Сергій Міняйлук. Об'єкт потенційно релевантний як **регіональний B2B-склад/шоурум для дистрибуції по Півдню України**, але: (а) компанія не демонструє публічних планів відкриття регіональних точок; (б) HQ і виробництво сконцентровані в Хмельницькому; (в) Одеса і Південь — потенційна ніша, але не підтверджена потреба. ## 6. «Крючки» — персональні аргументи для Velmet

Не загальні плюси об'єкта, а конкретні попадання під ситуацію Velmet:

**1. Логістичне «вікно» на Південь України**
Velmet виробляє в Хмельницькому і продає онлайн по всій Україні. У Одесі та Південному регіоні — 3+ активні театри бойових дій (Херсонська обл., Миколаївська обл., ПДНР). Одеський хаб-склад для держзамовлень і B2B-дистрибуції по Півдню — логічний крок для компанії з контрактами МВС і ЗСУ. Доставка зі складу на Дальницькій до точок на Півдні — швидша, ніж з Хмельницького.

**2. Ваш новий клас продукції — бронещити — потребує окремої логістики**
53 млн грн контракт МВС: 300 щитів по 30,5 кг. Це не спорядження для відправки «Новою Поштою». Окремий регіональний склад зі вантажними воротами 5 м і двором для розвантаження транспорту — профільна інфраструктура для таких поставок.

**3. Охоронена зона без додаткових витрат**
Продукція Velmet — держзамовлення ЗСУ і МВС. Вимоги до зберігання: охорона, відеоспостереження, огорожа. На об'єкті все це вже є: пожежна і охоронна сигналізація («Легіон»), відеоспостереження, паркан по периметру, закриті ворота. Не потрібно монтувати з нуля.

**4. Ставка $2/м² — нижче за хмельницький рівень**
Ринкова ставка виробничо-складських приміщень в Одесі — $2–3/м². Наша ставка на нижній межі ринку або нижче. Для регіонального хаб-складу 400–600 м² (частина будівлі) — $800–1200/міс. Це операційний B2B-актив з мінімальними витратами.

**5. Velmet Plastix і власні технології потребують місця для відпрацювання**
Компанія розвиває власні матеріали (ARM550, Plastix) і патентовані рішення. Окрема точка на Півдні може бути не тільки складом, а й майданчиком для тестування нових продуктів у «польових» умовах ринку.

**6. Двір 1028 м² для тестування стрілецьких мішеней**
Velmet виробляє металеві мішені для тирів і стрільбищ. Демонстраційний майданчик для B2B-клієнтів (тири, навчальні центри, силові структури) — унікальна опція, яку неможливо організувати в ТРЦ або офісній будівлі.
