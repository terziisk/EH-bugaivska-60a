Тому хочемо тримати вас у курсі — як опцію на майбутнє, без жодного натиску щодо строків.

В Одесі звільняється standalone-будівля 1500 м² у промзоні Хаджибейського району. Що може бути релевантним саме для виробника UGV і безпілотних систем:

- **Периметр безпеки «з коробки»:** огорожа по периметру (цементна основа + метал), закриті ворота, охоронна сигналізація (Легіон), відеонагляд, промислове оточення без щільної житлової забудови — стандартний пакет вимог для defense-виробника.
- **Закритий двір 1028 м²** — придатний для відкритих тестів наземної техніки на власній ділянці, без виїзду на зовнішній полігон; ворота 5 м без обмеження висоти для прийому компонентів і виходу прототипів.
- **Виробнича інфраструктура:** 50 кВт / 380 В (достатньо для складального цеху UGV + тест-стендів), декілька незалежних інтернет-провайдерів, цементна підлога складської частини, готова інженерія під складання та тестування.

У вкладенні — повна комерційна пропозиція з фото, планом ділянки, інженерними даними та концептуальними візуалізаціями фасаду під 8 різних форматів бізнесу. Ставка та умови договору — у вкладенні.
