## 4. Формат нерухомості

### Існуючі точки — профіль

- **Одеса (Катерининська/Європейська, 49):** центр міста, вулиця пішохідно-туристичного профілю. Це класичний street-retail у зоні з пішохідним трафіком — антипод нашого об'єкта (промзона + машинний трафік).
- **Київ (Почайна):** великий магазин поблизу метро, велоїзна зона всередині, парковка для велосипедів. Форматно ближчий до «флагмана». Площа точно не підтверджена, але очевидно 500+ м² за характеристиками.
- **Київ (Голосіївський):** точка у спальному районі, поблизу метро.

### Типові вимоги до приміщення (непряма оцінка за аналогами)

| Параметр | Оцінка |
|---|---|
| Площа стандартного магазину | 150–400 м² |
| Площа флагманського формату | 500–1200 м² (на прикладі Почайної) |
| Формат локації | Street-retail або ТЦ; переважно близько до пішохідного/велосипедного трафіку |
| Вимоги до підлоги | Рівна, витримує велосипеди + відвідувачів, бажано без порогів |
| Вимоги до висоти стель | Від 3 м (для підвішування велосипедів на стелажі) |
| Парковка | Велопарковка важлива; автопарковка — бажана |
| Двір / тест-зона | Ключове для флагманського формату (тест-їзда) |
| Електромощність | Стандартна (~10–20 кВт), немає спецвимог |
| Вантажний доступ | Не критично, але зручно для поставок велосипедів у коробках |

Примітка: вимоги до приміщень публічно не розкриваються. Оцінка на основі аналізу діючих магазинів і галузевих стандартів.

## 6. «Крючки» — персональні аргументи для Велопланети

### Чому об'єкт на Дальницькій 48 може зацікавити саме Велопланету

1. **Двір 1028 м² = тест-трек, якого немає ніде в Одесі.** Велопланета у Києві (Почайна) зробила ключову фічу з можливості «проїхатись всередині». В Одесі — жодна точка такого не має. Наш двір дозволяє організувати повноцінний тест-трек просто неба — унікальна конкурентна перевага для єдиного такого формату в Одесі.

2. **Автомобільна артерія — це цільова аудиторія велосипедистів-ентузіастів.** Клієнт, який купує Cannondale за $1500–4000, приїжджає до магазину на машині. Він не «заходить мимохідь», він «їде цілеспрямовано». Дальницька з трамваєм і щільним автопотоком — це зручно. Одеська точка на пішохідній Катерининській, навпаки, обмежена у форматі і площі.

3. **1500 м² = справжній флагман.** Нинішня одеська точка — за розміром явно компактна. 1500 м² дозволяє: окрема зона гірських, окрема — міських і дитячих; секція e-bikes; майстерня з очікувальною зоною; GURU Fit Studio; клубна/подієва зона. Це той формат, який Велопланета реалізувала на Почайній у Києві, — і якого в Одесі ще немає.

4. **Кліматизовані складські площі для зберігання сезонного стоку.** Велосипедний бізнес — сезонний (виробник Велотрейд прямо говорить про 5-кратний перепад обсягів). Задня частина будівлі ангарного типу (окремий складський блок) ідеально підходить під зберігання міжсезонного стоку без окремої оренди складу.

5. **Ворота 5 м + двір = прийом фур з велосипедами.** Велосипеди завжди доставляють у великих коробках / на паллетах. Наш двір приймає 5-10-тонник без виїзду на вулицю — для базового склад-магазину це критичний параметр.

6. **Ставка $2/м² при ринкових $11–12/м² для вулиць Одеси.** Об'єкт розташований поза центром, тому ставка кратно нижча. Для формату «велофлагман», куди їдуть цілеспрямовано — локація в промзоні не є мінусом, а ставка є прямим фінансовим виграшем.
