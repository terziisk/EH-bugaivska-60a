## 7. «Гачки» під об'єкт Дальницька 48

### Сильні аргументи

1. **Одеса = природна операційна база для морських дронів.** Випробування вже відбуваються в Чорному морі поблизу Одеси (підтверджено). Виробничий або складальний цех у 7 км від акваторії скорочує логістику між «зібрали» і «спустили на воду».

2. **Ворота 5 м без обмежень по висоті.** Корпус Magura V5 (5,5 м довжина, 1,5 м ширина) і V7 (7 м довжина, 2 м ширина) фізично проходять у ворота при перевезенні на причепі горизонтально. Двір 1028 м² дозволяє маневрувати з довгим причепом.

3. **Цементний підлогою та двір.** Масивні платформи Magura V7 (3 400 кг) — навантаження для підлоги. Цементний пол і відкритий двір дозволяють маніпуляції з важкою технікою.

4. **Охорона та безпека.** Відеоспостереження + охоронна сигналізація + паркан по периметру + ворота, що зачиняються — базовий пакет для defense-tech, який вже є.

5. **Електромощність 50 кВт / 380В.** Достатньо для складального цеху і зарядки акумуляторів морських дронів (не для масштабного серійного виробництва, але для R&D або mid-scale складання — так).

6. **Хаджибейський район = промзона.** Відсутність жилих будинків поряд — плюс для виробництва з шумовими і логістичними обмеженнями. Сусідство АЗС = постачання палива для двигунів.

### Слабкі сторони (чесно)

1. **Стеля 2,8–3,0 м.** Для збирання Magura V7 (2 м ширина, маса 3,4 т) при такій стелі знадобиться підйомне обладнання, яке майже не вписується. Це реальне обмеження для повноцінного складального виробництва.

2. **Масштаб компанії проти масштабу об'єкта.** UFORCE — єдиноріг із 1000+ людьми і 15 виробничими точками. Рішення про нові площадки ухвалюється на корпоративному рівні, не через звичайний outreach від орендодавця. Наш об'єкт може підійти як регіональний склад, R&D-майданчик або мала складальна точка — але навряд чи як флагманський завод.

3. **Локація виробництва засекречена.** Якщо в UFORCE вже є одеська точка — вони не скажуть про це публічно, і ми не можемо це перевірити.
