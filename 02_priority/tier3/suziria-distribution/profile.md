> 5000+ SKU, 30 ексклюзивних брендів). Мережа складів: Харків, Львів, Київська область (8000+ м²

> за профілем (лайт-хаб + офіс), але компанія швидше здатна орендувати власний сучасний склад,
> ніж переїхати в торговий об'єкт; потолок 2,8–3 м і відсутність висотного стелажування — обмеження.

## 2. Масштаб

- **Кількість B2B-клієнтів:** 6000+ зоомагазинів по Україні.
- **Асортимент:** 5000+ SKU, 30+ ексклюзивних міжнародних брендів.
- **Географія складської мережі:**
  - Повноцінні склади (розподільні центри): Харків, Львів, Київська область (с. Колонщина, 8000+ м², відкрито 09.2023, інвестиції 40+ млн грн).
  - Крос-докінг центри: **Одеса** (відкрито ~2023) та **Дніпро** (відкрито ~2024).
  - Загальна площа складської мережі (лише основні три склади): **21 300+ м²**.
- **Адреса одеського крос-докінгу:** Не підтверджено у відкритих джерелах.
- **Цільова аудиторія (B2B):** зоомагазини, ветеринарні клініки, pet-café, зоогрумінги.
- **Пропускна здатність:** ~10 000 замовлень/місяць по всій Україні.
- **Кількість співробітників групи:** 2200+.

| Висота потолків | Не підтверджено; РЦ мають WMS/TMS — ймовірно 6–8 м стандарт |
| Парковка / двір | Потрібні ворота для фур; двір під розвантаження |
| ТЦ vs стрит | Виключно окремостоячі складські/промислові об'єкти |
| Вимоги до електрики | Не підтверджено |
| Вимоги до пожежної | Стандартні складські |

**Ключове обмеження для нашого об'єкту:**
Основні РЦ Suziria — 8000+ м², вищі потолки. Для крос-докінг вузла (Одеса) 1500 м² теоретично реалістично, але потолок 2,8–3,0 м для крос-докінгу зоотоварів (коробки, паллети) є пороговим: зоотовари — нетяжкі, паллети 1–2 яруси, гідравлічні рохлі — цілком підходять. Двір на одну фуру є обмеженням: крос-докінг вимагає 2–4 одночасних заїзди в ідеалі. Але для малого одеського вузла (не головний РЦ) — прийнятно.

## 6. «Крючки» — персональні аргументи

1. **Крос-докінг вузол Одеси потребує адреси.** Поточний одеський крос-докінг відкрито ~2023; конкретна адреса публічно не розкривається. Якщо об'єкт орендовано у невеликого гравця або на тимчасових умовах — Дальницька 48 є природним апгрейдом: промислово-комерційна зона, 5-метрові ворота, власний двір під розвантаження, цементна підлога, 50 кВт.

2. **Зоотовари — легкі.** Потолок 2,8–3,0 м не є блокером для зоотоварів: суха продукція, корми, аксесуари в коробках. Паллети не потребують висотного стелажування — 1-2 яруси рохлями. Наш об'єкт фізично справляється.

3. **Двор = швидке сортування.** 1028 м² вільної зони — це маневровий двір для 1–2 малих вантажівок + легкові автомобілі менеджерів і торгових представників. Крос-докінг для Одеси і Півдня (Миколаїв, Херсон) — невеликий за обсягом, і одного маневрового двору достатньо.

4. **Адреса для MasterZoo.** Якщо Suziria Distribution переїде на Дальницьку, об'єкт автоматично стає логістичним хабом для **поточних і нових MasterZoo-магазинів Одеси** (3 стандартні + плановий флагман 500+ м² у 2026). Склад дистрибуції і новий флагман MasterZoo можуть бути в одному адресному кластері — нетипова пропозиція для групи.

5. **Ставка $2/м² vs. ринок.** Для об'єкту 1500 м² у промзоні ставка $3000/мес — це $2/м². Ринок складської оренди Одеси клас B — $4/м², клас A — $5–6/м². Тобто ставка вдвічі нижча за ринок. Для операційного підрозділу групи (не роздріб, а логістика) це пряма економія $24–36К/рік порівняно з ринком.

6. **Безпека вже є.** Suziria Distribution зберігає ексклюзивні товари провідних брендів — безпека критична. На Дальницькій 48 вже є охоронна сигналізація, відеоспостереження, огорожа по периметру, закриті ворота. Не потрібно вкладати в security-інфраструктуру.

### Важливе застереження

Зв'язок Suziria Distribution з об'єктом — непрямий. Компанія активно інвестує у власну складську інфраструктуру, не в орендоване майно.
