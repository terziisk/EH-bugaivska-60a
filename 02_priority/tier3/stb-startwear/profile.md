# STB / Саксагань-Торгбізнес — профіль компанії

**Резюме (4–5 рядків):**
STB (Saksagan-Torgbiznes) — швейне підприємство малого масштабу з Кривого Рогу, що спеціалізується на пошитті тактичного та військового спорядження, форми і спецодягу за ТУ Міноборони. ISO 9001 сертифіковане виробництво. 3 цехи, 80 швей, 7 лазерних розкрійних столів. За реєстром — 1 офіційно оформлений співробітник, що вказує на характерну для малого бізнесу модель ФОП/субпідряду. Штаб і виробництво знаходяться в Кривому Розі — прифронтовому місті (в радіусі авіаударів з 2024 р.), що є гіпотетичним мотивом для диверсифікації. Проте публічних планів розширення або виходу в Одесу не виявлено. Фіт з об'єктом — **низький (3/10)**: об'єкт технічно підходить, але STB — занадто малий гравець, щоб вибирати самостійне будівлю 1500 м² за 3000–3500 $/міс.

## 6. «Крючки» — що саме зачепить STB

1. **Кривий Ріг — під ударами, Одеса — не під ударами.** Атаки на Кривий Ріг у 2024–2025 рр. не зупиняються. Для виробника, що постачає ЗСУ і не може дозволити собі зупинку виробництва, «другий майданчик у відносно захищеному портовому місті» — страховий аргумент. Об'єкт у Хаджибейському районі (промзона без житлових кварталів навколо) — менш приваблива ціль, ніж центр міста.

2. **ISO 9001 + ТУ МО → постійні держзамовлення → потреба у стабільному майданчику.** Виробник, що отримує держзакупівлі, несе відповідальність за строки та обсяги. Зупинка виробництва через вимушену релокацію — ризик штрафних санкцій. Готовий майданчик із інженерією «під ключ» (50 кВт, водопостачання, каналізація, охорона) скорочує downtime до мінімуму.

3. **Одеський порт = логістика для експорту.** На сайті немає згадок про експорт, але якщо компанія постачає спорядження для міжнародних закупівель (НАТО-стандарти, волонтерський ринок), Одеса дає прямий вихід на морський і залізничний хаб для відвантаження на Захід.

4. **Ставка $2/м² при ринковій $2,5–4/м² для виробничих приміщень Кривого Рогу і Одеси.** 1500 м² за $3000/міс — нижче за ринок. Для виробника з маржею B2B-підряду різниця $500–1000/міс помітна.

5. **Об'єкт приймає газелі і малотоннажні фури (до 10 м) без проблем.** Типовий «матеріальний» цикл швейного виробника: рулони тканини (40–300 кг, газель або 5-тонник) + відвантаження готового спорядження в коробах. Двір 1000 м² + ворота 5 м = маневр без перекриття вулиці.
