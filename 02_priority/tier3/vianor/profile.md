# Vianor (TOV Nokian Shina) — профиль компании

## 1. Базовое

| Параметр | Значение |
|---|---|

Nokian Tyres — крупнейший производитель зимних шин в мире; Vianor — принадлежащая ему розничная шинная сеть. В Украине работает **исключительно по франшизе** — все центры принадлежат украинским предпринимателям. ТОВ «Nokian Shina» выступает правообладателем концепции, поставщиком шин и координатором сети.

## 4. Формат нерухомості

| Параметр | Вимоги Vianor |
|---|---|
| Тип об'єкта | Стрит-ритейл, «червона лінія», поруч з АЗС або автосалоном |
| Площа | Від 150–200 м² (Express — 1 пост) до 500–1000 м² (повний формат, 4–8 постів) |
| Висота стель | Не менше 3,5–4 м (підйомник або яма); в нашому об'єкті стеля 2,8–3,0 м — РИЗИК |
| Ворота | Обов'язково; ширина 3–4 м мінімум |
| Парковка | Мінімум 2–4 машино-місця перед входом |
| Фасад | «Червона лінія», видимість з дороги |
| Електромощність | 380V — обов'язково (підйомники, компресор, балансувальний стенд) |
| ТЦ чи стрит | Завжди стрит; ТЦ не відповідають сервісному формату |

**Критична обмеженість об'єкта:** Vianor вимагає ями або підйомників (мінімум 1–2 підйомника). При стелі 2,8–3,0 м стандартний двостійковий підйомник (потрібна висота ~4,0–4,5 м) не встановити. Альтернатива — ножичний підйомник або яма, яких в об'єкті немає. Це **головне технічне обмеження** для розміщення класичного шиномонтажу в нашому об'єкті.

**Що реалістично:** формат без підйомника — тільки шинний магазин/шоурум + склад + видача шин на зберігання + дрібний сервіс (підкачка, азот). Або переговори про облаштування ями — потрібно уточнити у власника.

## 6. «Крючки» — персоналізовані аргументи

### Чому наш об'єкт може зацікавити Vianor (конкретно, не загально)

**1. Консолідація / апгрейд одеської мережі**
Vianor в Одесі — 7–9 малих точок площею 150–400 м² у різних районах. Кожна точка в «мінімальному» форматі: 1–2 пости, обмежений склад, немає зони розширеного асортименту (диски, автохімія, аксесуари). Наш об'єкт 1500 м² + двір 10 соток — це можливість відкрити **«Vianor Hub Odessa»**: шинний магазин повного асортименту + сезонне зберігання шин на 300–500 комплектів + шоурум дисків і автохімії Nokian + корпоративний сервіс (Vianor Club). Це не конкуренція існуючим точкам, а новий рівень сервісу.

**2. Сезонне зберігання шин — дефіцитна послуга в Одесі**
У всіх одеських точках Vianor немає власного складу під зберігання — клієнти зберігають шини «під навісом» або відмовляються від послуги. Наш двір 1028 м² + ангарна частина = повноцінний склад зберігання. Це окремий дохідний продукт і утримання клієнтів.

**3. Ворота 5 м + двір без обмежень по висоті**
Завезення великих партій шин (піврічна поставка від Nokian) і прийом клієнтів з SUV/мінівенами та позашляховиками — потрібна зона маневру. Наш двір приймає фуру до 10 м і дає можливість обладнати зону розвантаження окремо від клієнтської парковки.

**4. АЗС поруч — готовий автомобільний трафік**
Дальницька — головна виїзна артерія Одеси з постійним автопотоком. АЗС безпосередньо поруч = клієнти, які вже зупинились і звернули увагу. Це точно «автомобільний» трафік, не пішохідний.

**5. Двір як відкритий шоурум дисків**
1028 м² вільної огородженої території — можливість виставляти диски та сезонну автохімію «живцем» просто неба, що в стандартному шиномонтажі 150 м² неможливо. Ефект придорожнього авторинку.

**6. 380В/50 кВт — всі підйомники без обмежень**
Електрики вистачить на 3–4 стійкових підйомника + компресор + балансувальний стенд + освітлення.

**ВАЖЛИВО — технічне обмеження, яке треба проговорити:**
Стеля 2,8–3,0 м у торговій частині блокує стандартний двостійковий підйомник. **Рішення:** (а) уточнити, чи можна облаштувати 1–2 ями у ангарній частині (власник має підтвердити); (б) розглянути ножичний підйомник (висота підйому ~1,5–1,8 м, вміщується при стелі 3 м); (в) сервісна зона тільки для ям + шиномонтаж без підйому — достатньо для шиномонтажу та балансування.
