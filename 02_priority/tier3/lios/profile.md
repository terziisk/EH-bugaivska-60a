## 6. «Гачки» — чому наш об'єкт релевантний саме для LIOS

**6.1. Об'єкт був меблевим шоурумом BEST Меблі (~2000 м² виставкової площі) — концепція доведена**

LIOS не буде «першопрохідцем» у нестандартному місці. Будівля вже має меблевий track record: власники шоурумів м'яких меблів розуміють цю мову. «Ваш тип бізнесу вже успішно працював у цьому приміщенні» — це не pitch, це факт.

**6.2. Перехід від «секції в меблевому ТЦ» до власного брендового шоуруму**

Зараз одеська точка LIOS — секція всередині гіпермаркету «Шостий елемент» на Овідіопольському шосе. Це означає: немає окремої вивіски, немає власної парковки під бренд, немає контролю над оточенням (поруч конкуренти). Наш об'єкт пропонує окремостояче будівля з 28 м фасаду, яке повністю належить бренду орендаря.

**6.3. 28 метрів вітринного скління = вітрина для диванів просто з вулиці**

Для м'яких меблів вітрина — основний інструмент продажу: клієнт повинен бачити дивани й крісла ще до входу. 28 м фасаду з вітринним склінням на всю висоту дають змогу розмістити 4–6 повноцінних інтер'єрних «сцен» (кутовий диван + кавовий столик + торшер), видимих з вулиці та з автомобіля на ходу.

**6.4. Двір 1028 м² — рідкісне «outdoor studio» для меблевого бренду**

Можливе розміщення виставкових зон на відкритому повітрі: веранда-меблі, терасні дивани, садові крісла. Для B2B-клієнтів (ресторани, готелі, яхт-клуби Одеси) — наочна демонстрація «як це виглядатиме на вашій терасі». Цей аргумент особливо актуальний для Одеси як приморського міста з розвиненим ресторанним та готельним ринком.

**6.5. Можливість поєднати шоурум + виробничий/складський хаб для півдня України**

Якщо LIOS розглядає регіональну логістику для Одещини та Причорноморського регіону, наш об'єкт дозволяє поєднати у одній будівлі: шоурум (фасадна частина з вітриною), склад готової продукції для доставки по регіону (тильна ангарна частина), можливо — невеликий збиральний цех для дошиву та підгонки під замовлення клієнтів. Ворота 5 м + двір дозволяють вільно приймати і відвантажувати великогабаритні меблі.

**6.6. Ставка $2/м² при ринкових $11–12/м² для street-рітейлу**

Перехід з секції у «Шостому елементі» (орієнтовні ставки меблевих ТЦ: $8–15/м² залежно від формату) до окремостоячого шоуруму 1500 м² за $3000–3500/міс — це або рівнозначна вартість, або дешевше при більшій площі.
