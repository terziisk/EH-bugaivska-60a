# Swarmer Inc. — профіль компанії

> **Summary (3–5 рядків):** Swarmer — перша українська defense-tech компанія, що вийшла на Nasdaq (березень 2026, тікер SWMR). Розробляє AI-ПЗ для координації роїв дронів (платформа Styx), **не виробляє дрони фізично**. Операційна команда R&D (~40+ розробників) базується в Київі та Львові, не в Одесі. Офіс в Одесі не підтверджений жодним джерелом. Фіт з об'єктом — теоретично сильний (охорона, забор, 50 кВт, окрема будівля), але реальна потреба у приміщенні в Одесі зараз не задокументована. Рекомендований підхід: outreach як «можливість для другого одеського хабу» до VP Operations і Director of HR.

Swarmer — переважно **software company**, а не фізичне виробництво. Тому вимоги до приміщень суттєво відрізняються від класичних drone-manufacturers:

| Параметр | Потреба Swarmer | Наш об'єкт |

| Тип приміщення | Software R&D офіс + hardware integration lab (тест-стенди, дрони) | Торгово-офісний + лег. склад — конвертується |
| Площа | Не розкривається. При ~40 розробниках в UA — орієнтовно 600–1200 м² | 1500 м² (гнучка планування, ГКЛ-перегородки) |

| Охорона та периметр | Критично важливо (секретні дані, обладнання) | Охоронна сигналізація «Легіон», відеонагляд, паркан, ворота — повна відповідність |

## 6. «Гачки» — чому саме цей об'єкт для Swarmer

1. **Охорона = базова вимога, а не опція.** Swarmer — публічна компанія на Nasdaq з секретними бойовими алгоритмами та обладнанням. Охоронна сигналізація «Легіон» + відеонагляд + паркан по периметру + закриті ворота — це готова інфраструктура фізичної безпеки. Більшість офісних БЦ Одеси цього не дають.

2. **Кілька незалежних провайдерів інтернету** — для R&D команди з розподіленими системами (Styx AI, edge computing на дронах) резервування каналу критично. На об'єкті це є «з коробки».

3. **Промзона без зайвих очей.** Тест-польоти FPV-дронів у закритому дворі 1028 м² в промзоні Хаджибейського р-ну — це можливо і не привертає уваги. У центрі міста або в ЖК — неможливо.

4. **Окрема будівля зі своїм двором** = власний «campus». Після IPO Swarmer будує бренд публічної компанії — standalone-будівля в Одесі з вивіскою та огородженою територією дає правильний корпоративний образ на відміну від орендованого поверху в БЦ.

5. **Ставка $2/м² при ринку $5–8/м² для офісу** — для публічної компанії, яка звітує перед інвесторами, суттєва економія на OPEX. Різниця ~$4000–9000/міс = $48–108К/рік.

6. **«Одеський крок» у межах стратегії.** Swarmer заявив розширення в Київ, Львів, Варшаву і Японію. Одеса — портове місто з активним IT-кластером та прямим зв'язком з морським theater of operations. Якщо компанія думає про тестування морських автономних систем (naval drones) — Одеса логічна.

## ⚠️ Ключові застереження

1. **Swarmer — software company, не drone manufacturer.** Їм потрібен офіс + hardware lab, а **не** склад чи виробничий цех у класичному розумінні. Об'єкт відповідає, але pitch повинен бути «офіс + закрита тест-зона», а не «виробниче приміщення».

2. **Одеського офісу зараз немає.** Жодне підтверджене джерело не фіксує офіс Swarmer в Одесі. Outreach — це пропозиція **нового** хабу, а не розширення існуючого.

3. **Зв'язок з Odessa Journal** у назві статті [«Ukrainian startup Swarmer has filed for IPO»](https://odessa-journal.com/the-ukrainian-startup-swarmer-has-filed-for-an-ipo-on-nasdaq) — не означає, що компанія одеська. Odessa Journal публікує загальноукраїнські новини.

4. **Fit score 4/10 (а не 5/10)** через відсутність підтвердженої потреби в одеському приміщенні. При з'явленні сигналів (вакансія в Одесі на DOU, публічна заява про одеський офіс) — переглянути на 5/10.
