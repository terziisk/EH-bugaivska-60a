## 4. Формат нерухомості

- **Поточний формат:** офісно-виробниче приміщення на вул. Б. Хмельницького, 24 в Одесі. Це центральна-пів-центральна частина міста. Точна площа не розкривається публічно. За форматом (повний виробничий цикл: зварювання, фрезерування, друк, монтаж + офіс + штат 20–40 осіб) — орієнтовно 400–800 м². Публічних даних про власну нерухомість немає — ймовірно, оренда.
- **Вимоги до воріт та висоти:** виробництво великих рекламних конструкцій (вивіски від 1 до 10+ м, пілони, стели, металеві каркаси, намети і тенти) вимагає воріт не менше 4–5 м шириною та необмеженої висоти для вивезення готових конструкцій. Об'єкт: **ворота 5 м без обмеження по висоті** — точне попадання.
- **Вимоги до двору:** зварювальний цех потребує майданчика для збирання великих конструкцій перед монтажем і для зберігання металопрофілю, рулонів плівки, балок. Двір 1028 м² — оптимальний формат.
- **Вимоги до підлоги:** зварювальний та фрезерний цех потребує бетонної/цементної підлоги підвищеної міцності. Об'єкт: цементна підлога у складській частині — повне попадання.
- **Вимоги до електрики:** УФ-друк, плоттери, зварювальне обладнання, фрезерні верстати з ЧПУ + кліматизація = мінімум 30–50 кВт. Об'єкт: 380В / 50 кВт — відповідає. При необхідності можлива розмова про збільшення мощності (відкрите питання з облєнерго).
- **Вимоги до вентиляції:** зварювання металу та УФ-лакування/друк вимагають потужної витяжної вентиляції. Об'єкт: можливе встановлення приточно-витяжної вентиляції з виводом на вулицю.
- **Вимоги до висоти стель:** зварювальний цех для великих конструкцій ідеально потребує 4–6 м. У об'єкта — 2,8–3,0 м у торговій зоні та ~3 м у складській частині ангарного типу. Це є обмеженням для збирання дуже великих металоконструкцій (вертикальні пілони 5+ м). Для горизонтального виробництва (вивіски, лайтбокси, стенди, фасадні елементи) — достатньо.
- **ТЦ або стрит:** виробнича модель, не ритейл. Окремо стояча будівля зі своїм двором — оптимальний формат для такого бізнесу. Локація в промзоні Хаджибейського р-ну — органічна для виробництва реклами.
- **Вимоги до парковки:** клієнти-корпоранти і монтажні бригади приїжджають на авто/фургонах. 6–7 місць + вулична парковка — прийнятно.
- **Вимоги до фасаду:** для виробника реклами власний впізнаваний фасад є одночасно рекламою та шоурумом. Фасад 28 м прямо на магістраль Дальницька — ідеальна можливість для демонстрації власних виробів (вивіски, LED-конструкції, нестандартні фасадні рішення).

## 6. «Гачки» — персональні аргументи для Фабрика реклами

### Гачок 1 (головний): двір 18 соток + ворота 5 м без обмеження висоти — мрія рекламного виробника

Виробництво великих рекламних конструкцій — вивіски, пілони, стели, намети, фасадні системи — фізично обмежене простором двору та висотою воріт. Поточне приміщення на вул. Б. Хмельницького, 24 у центральній частині Одеси майже напевне не має великого виробничого двору: центральна забудова = щільний периметр.

Об'єкт на Дальницькій 48: **двір 1028 м² + ворота 5 м без обмеження по висоті** — це:
- Майданчик для збирання великих металоконструкцій (пілони, арки, торговельне обладнання) перед вивезенням на монтаж
- Зберігання металопрофілю, труб, панелей, рулонів банерної тканини під відкритим небом (під плівкою/навісом)
- Вільний заїзд вантажного транспорту (до 10-тонника) для доставки великогабаритних матеріалів і відвантаження готових конструкцій без виїзду на вулицю

Для рекламного виробника такого масштабу це апгрейд якості роботи, а не просто «більше квадратів».

### Гачок 2: утроєння штату = переросли поточні площі

Сайт компанії сам повідомляє: штат «утроївся» з моменту заснування. Якщо 20 років тому стартували 5–10 осіб — зараз це 20–40 чоловік на одному виробничому майданчику. При повному виробничому циклі (зварювання + фрезерування + друк + монтаж + офіс дизайнерів + склад матеріалів + склад готової продукції) — стандартні 400–800 м² у центральному районі стають тісними. Зростаючий ринок OOH (+25% у 2025, +17% прогноз 2026) тільки збільшує навантаження.

1500 м² на Дальницькій — можливість нарешті розмежувати зони: зварювальний цех / фрезерний цех / цех широкоформатного друку / монтажний ангар / склад матеріалів / склад готової продукції / офіс та шоурум клієнтів.

### Гачок 3: промзона на трасі — органічна локація для рекламного виробника

Поточна адреса — вул. Б. Хмельницького, центральна частина Одеси. Це «не той» тип локації для важкого виробництва: шумне зварювання, вантажний транспорт, великогабаритні конструкції — все це давить на сусідів та міську інфраструктуру.

Хаджибейський район (вул. Дальницька) — 42% індустріальних підприємств міста. Це **нормальна локація для рекламного виробника**: сусіди — СТО, склади, виробничі підприємства. Жодних обмежень по шуму та вантажному трафіку. Переїзд у промзону для рекламного цеху — це не «деградація», а відповідність формату бізнесу.

### Гачок 4: фасад 28 м на магістралі — живий шоурум власного виробництва

Будь-яка рекламна компанія заробляє репутацію через видимість власних виробів. Кращий «кейс-портфоліо» — це сам фасад будівлі, де вони працюють.

Фасад 28 м прямо на вул. Дальницька з трамваєм та автомобільним потоком = **щоденний шоурум на головній виїзній магістралі Одеси**. Фабрика реклами може оформити фасад так, як жодна інша одеська рекламна компанія не може собі дозволити: власні LED-вивіски, нестандартні об'ємні літери, зразки різних типів конструкцій прямо на фасаді будівлі. Це маркетинг без маркетингового бюджету.

### Гачок 5: ставка $2/м² — у 5–6 разів нижче street-retail, але цей об'єкт не street-retail

Орендна ставка 3000–3500 USD/міс. (~$2/м²/міс.) для виробничо-складського формату в Одесі — **рыночна або нижче ринку** (виробничі приміщення в Одесі: $2–3/м²). Порівняно з поточними витратами на аренду в центральній частині міста (вул. Б. Хмельницького) — можливо вигідніше або порівняно, при значно більших площах і ліпшій інфраструктурою.
