## 4. Формат нерухомості

- **Тип об'єктів:** виробничо-складський комплекс (пров. Хвойний, 8б) + офіс/поштова адреса (вул. Щорса, 83). Обидва в Хаджибейському районі — промислово-комерційна зона
- **Вимоги до виробничого приміщення (виведено з профілю виробництва):**
  - Офсетний друк потребує стабільного живлення 380В, кліматконтролю (вологість/температура), надійної підлоги
  - Склад паперу та готової продукції: цементний або наливний підлогу витримує рохлі та паллети
  - Прийом рулонного паперу та картону: необхідний вʼїзд для вантажних авто (мінімум Газель/ГАЗель-3т, реально потрібен 5-тонник)
  - Площа: 44 співробітника + 5 регіонів координації → потреба в 600–1200 м² виробничих + 200–300 м² офісних + склад готової продукції. Сукупно 1000–1500 м²
  - Зона відвантаження: виробництво відвантажує партії пакетів/коробок у регіони через НП/Meest, щоденне завантаження
- **Потреба у стелажах:** висота стелажів обмежена (паперові пакети та картон — відносно лёгкий і не дуже високий товар), потолки 2,8–3,0 м є прийнятними
- **Паркування:** необхідна ділянка для відвантаження Газелей/мінівенів, стандартна паркова зона для 5–10 авто
- **Вимоги до фасаду:** не є критичними (виробнича компанія, не роздрібний трафік). Адреса для корпоративних клієнтів важливіша, ніж видимість з дороги
- **Потреба у вентиляції:** офсетний друк з розчинниками (лаки, фарби) потребує приточно-витяжної вентиляції. Наш об'єкт допускає монтаж вентиляції (технічно підтверджено в специфікації)

## 6. Гачки під Gift-K

**Головний гачок — консолідація двох адрес в одну:**

Gift-K має два окремі об'єкти в Одесі (пров. Хвойний, 8б — виробництво; вул. Щорса, 83 — офіс/склад). Обидва в Хаджибейському районі, відстань між ними — орієнтовно 1–1,5 км. Компанія фактично несе витрати на дві оренди, дві комунальні плати, логістику між локаціями. Наш об'єкт (1500 м² + двір 18,2 сотки) дозволяє об'єднати виробництво + офіс + склад готової продукції в одному місці з суттєвою операційною економією.

**Специфічні гачки:**

1. **«Одна адреса замість двох — без доплати»**: наша ставка 3000–3500 USD/міс за 1500 м² — при консолідації це може виявитися дешевше або порівнянно з двома окремими орендами. Аргумент: «ви вже платите за дві адреси — порівняйте»

2. **Цементний підлог і двір для вантажних авто**: виробництво Gift-K щодня отримує папір і відправляє готову продукцію через перевізників. Власний двір (1028 м²) + ворота 5 м + в'їзд для 5-тонника без виїзду на вулицю — це операційна перевага, якої немає на більшості міських адрес

3. **380В/50 кВт на борту**: офсетний друк, різальне та ламінувальне обладнання вимагає трифазного живлення. Об'єкт вже підключений до 380В — без витрат на підключення

4. **Вентиляція — монтаж можливий**: офсетний друк із лаками та фарбами потребує приточно-витяжної вентиляції. Конструктив об'єкту дозволяє вивести вентиляцію на вулицю через тильну стіну або дах ангарної частини

5. **Шоурум зразків в одному будинку з виробництвом**: вітринний фасад 28 м з суцільним остекленням — унікальна можливість для типографії зробити «живий шоурум» упаковки: клієнти приїжджають, бачать зразки, тут же замовляють. Для B2B-типографії, де клієнти — ювелірні мережі, ресторани, ритейлери — це реальна продажна перевага

6. **Хаджибейський район — «своя» промзона**: компанія вже 20+ років в цьому районі. Переїзд «залишається в знайомому середовищі» — мінімальний стрес для персоналу та логістичних партнерів

7. **Сусід — New Media (500 м)**: найпотужніша конкуруюча типографія півдня України вже на Дальницькій, 25/3. Це підтверджує, що локація органічна для полiграфічного виробництва
