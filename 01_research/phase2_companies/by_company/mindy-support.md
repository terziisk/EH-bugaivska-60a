## Summary

Mindy Support — один из крупнейших BPO-операторов Украины (data annotation, customer support, AI-тренинг), 2000+ сотрудников, 8 офисов по всей стране. В Одессе офиса нет — это явное окно для outreach. Компания последовательно открывает новые офисы в регионах (последние: Sumy, Vinnytsia, Zhytomyr), стандартный формат — 200–500 м² в коворкинге или отдельном помещении под операторский зал. Объект Дальницкая 48 (1500 м², отдельностоящее здание, охрана, несколько провайдеров) кратно превышает их типовой офис, поэтому реалистичный сценарий — либо субаренда части здания (~300–500 м²), либо позиционирование объекта как "флагманского хаба для Юга Украины". Фит — средний (5/10): потенциал есть, но нужно нестандартное предложение.

## 4. Формат недвижимости

| Параметр | Требование / оценка |
|---|---|
| Тип объекта | Офисное помещение в деловой/полуделовой зоне. Коворкинг или отдельный этаж/здание |
| Типовая площадь на локацию | ~300–600 м² (оценка, не підтверджено официально) |
| Рабочие места | 100–300 операторов на офис; плотность 4–6 м²/место |
| Требования к ремонту | Современный офис: open space + переговорки + зона отдыха + кухня |
| Электромощность | Достаточно стандартного офисного уровня; много ноутбуков/ПК, IP-телефония, видеонаблюдение |
| Интернет | Критически важно: несколько провайдеров или резервный канал (стандарт BPO для uptime SLA) |
| Парковка | Не критично (сотрудники — молодёжь, частично студенты, добираются общественным транспортом) |
| Высота потолков | Стандартная офисная (2,7–3,2 м) |
| Безопасность | Важно: физическая охрана, CCTV, контроль доступа (ISO 27001 обязывает) |
| Локация | Предпочтительно: центр города или транспортный узел с ОТ; промзона — нетипично, но возможно при наличии трамвайного маршрута |
| ТЦ или стрит | Стрит, отдельное здание или бизнес-центр |

## 6. "Крючки" — персональные аргументы для нашего объекта

**Почему именно Mindy Support — не просто BPO:**

1. **Одесса = единственный мегаполис Юга без вашего офиса.** 8 городов по всей Украине, но Юг — белое пятно. Одесса (~920К жителей, ОНУ + Политех + ещё 12 университетов) — незакрытый резервуар для найма операторов и аннотаторов. Конкуренты (Helpware, SupportYourApp) туда уже смотрят.

2. **Несколько провайдеров из коробки — это ваш SLA.** Для BPO с клиентами из Fortune 500 стандарт — uptime 99.9%. В нашем здании к каждому этажу подведены независимые провайдеры. Резервирование канала — не capex, а данность. Это прямо закрывает пункт ISO 27001 по business continuity.

3. **ISO 27001 + охрана + забор + видеонаблюдение — всё уже есть.** Для сертифицированного BPO физическая безопасность помещения — обязательное требование при ресертификации. В нашем объекте пожарная сигнализация, охранная система "Легіон", CCTV, огороженная территория с воротами — ни один коворкинг такого не даёт.

4. **Отдельностоящее здание = гибкость планировки под ваш стандарт.** Перегородки из гипсокартона — демонтируемые. Хотите open space на 200 операторов? Переговорные секции? Зона отдыха с Xbox? Собственный брандированный коридор на входе? Всё реализуемо без капительных работ.

5. **Промзона с трамваем — не минус для BPO.** Ваши операторы в других городах (Ternopil, Poltava) — молодёжь, которая ездит общественным транспортом. Ул. Дальницкая — трамвайная линия прямо у объекта. Для сотрудников, которые не приезжают на машине, это идеальная точка входа.

6. **Ставка $2/м² при рыночных $5–8/м² за офис — это экономия ~$4500/мес** для помещения 600 м² (примерный размер вашего одесского офиса). За 3 года контракта — $162K сэкономлено только на аренде.
