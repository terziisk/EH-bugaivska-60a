# Intellias — Профіль компанії для комерційної пропозиції

> Summary (5 строк): Intellias — одна з топ-10 IT-компаній України (~3000 спеціалістів, 14 країн), заснована у Львові в 2002 р. Одеський офіс існує з 2016 р. і розміщений у центрі міста на вул. Леха Качинського, 7 — в бізнес-класовому приміщенні в районі Грецькій площі. Компанія прогнозує зростання 10–20% у 2025 р. та активно наймає (+450 спеціалістів за початок 2025). Стратегія Intellias — «менше офісів, але сильніші»; нових відкриттів в Україні не заявлено, але зростання штату означає потенційну потребу у розширенні приміщень. Фіт зниженій-середній: компанія вже має одеський офіс, але 1500 м² на Дальницькій є нетиповою локацією для IT; шанс — якщо в Одесі назріло розширення або консолідація підрозділів.

## 4. Формат нерухомості

| Параметр | Значення |
|---|---|
| Тип приміщень | Бізнес-центри класу B+/A або standalone-будівлі в центральних/субцентральних локаціях |
| Поточна одеська локація | БЦ у центрі (вул. Леха Качинського — центр міста) |
| Вимоги до інтернету | Мінімум 2 незалежних провайдери, резервні канали — критично для IT |
| Вимоги до безпеки | Охорона, СКУД, відеоспостереження — стандарт |
| Вимоги до паркування | Невисокі: IT-спеціалісти їдуть на ОТ/велосипеді/таксі. Для управлінців — 5–10 м/м |
| Вимоги до висоти стель | Стандарт офісу: 2,8–3,2 м (наш об'єкт — відповідає) |
| Електромощність | 50–100+ кВт для офісу 100–200 осіб (в т.ч. серверна, кондиціонування) |
| Вентиляція | Припливно-витяжна або клімат-система — обов'язково |
| Характер простору | Open space + переговорні кімнати + зони релаксу + кімната для дітей (є у Lviv/Kyiv офісах) |
| Підвал/укриття | ⚠️ Критичний фактор: всі нові офіси Intellias мають власне укриття або підвал. Для Одеси — теж обов'язково |

**Коментар по об'єкту:** Дальницька, 48 — промзона, не центр. Для Intellias це нетипова локація (сучасний IT-офіс зазвичай відкритий в центрі або субцентрі з іміджевими причинами: залучення та утримання талантів). Але standalone-будівля зі своєю територією і охороною може бути аргументом для back-office / operations hub, не front-facing офісу розробки. Потолок 2,8–3,0 м — ОК для офісу. Підвального укриття на об'єкті немає (не зазначено в специфікації) — це потенційний стоп-фактор.

**Джерело:** [Intellias — Where we are](https://career.intellias.com/where-we-are/), [DOU Ревізор у Львові](https://dou.ua/lenta/articles/dou-revisor-intellias/) (опис офісного стандарту), аналіз специфікації об'єкта

## 6. «Крючки» — персональна релевантність для Intellias

**Чому Intellias може бути зацікавлений в об'єкті:**

1. **Зростання штату в Одесі без нового офісу** — компанія наймає +450 спеціалістів у 2025, частина з них в Одесі. Якщо поточний офіс на Качинського переповнений — standalone-будівля 1500 м² це спосіб вирішити питання «оптом», не шукаючи допофіс у БЦ.

2. **Автономна будівля зі своєю охороноюю і СКУД — це безпековий стандарт IT** — в умовах воєнного стану Intellias налаштована серйозно на безпеку (підвали в нових офісах Lviv/Kyiv). Об'єкт має периметр, ворота, охоронну сигналізацію, CCTV. Можна дообладнати укриттям.

3. **Кілька незалежних провайдерів інтернету — вже підведені** — це рідкість навіть для БЦ. Для IT-компанії резервування каналів критично.

4. **Ставка $2/м² — це в 2–4× нижче ринкового офісу класу B в Одесі ($5–8/м²)** — на 1500 м² економія 4500–9000$/міс відносно стандартного офісного ринку. Для команди 120–150 осіб це суттєво.

5. **Гібридна планировка** — зони торгівлі/офісу/складу переробляються в IT-простір: open space + переговорні + серверна + укриття (підвал або окрема кімната). Перегородки — гіпсокартон, демонтуються без капітальних робіт.

6. **Модель «back-office hub»** — Intellias має front-facing офіс у центрі (Качинського) для залучення талантів та зустрічей з клієнтами. Дальницька може стати back-office / operations / DevOps hub. Аналог прийнятий у великих IT-компаній: «парадний» офіс у центрі + операційний hub у промзоні з дешевшою орендою.

7. **Окремий двір для івентів / корпоративів / team building** — 1028 м² відкритої території при будівлі. Це нетипово навіть для великих БЦ в Одесі.
