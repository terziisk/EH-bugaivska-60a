# Accord Import — Профіль компанії

> **Короткий висновок (5 рядків):** Accord Import — найбільший в Україні експортер корпусних меблів, постачальник JYSK на 70% своїх замовлень. Виторг 2025 — 2,7 млрд грн (~$65M), контракт з JYSK на €80M готується до підписання у травні 2025. Будують 2-й завод за $14M у Хмельницькому, запуск — вересень 2025. Весь цикл орієнтований на виробництво та експорт: 96% продукції іде до ЄС. Для об'єкту на Дальницькій 48 Accord Import цікавий як потенційний орендар під регіональний шоурум-склад (південь України та B2B-дистрибуція), але публічних планів на одеський хаб не виявлено — гіпотеза потребує підтвердження.

## 4. Формат нерухомості

**Профіль приміщення:**
- Обидва заводи — власна нерухомість у Хмельницькому (не оренда).
- Компанія будує, не орендує виробничі потужності — це принципова позиція для основного виробництва.
- Для **регіональних складів та шоурумів** публічних даних про стратегію (власність чи оренда) не знайдено.

**Технічна інфраструктура поточного заводу:**
- Сонячна електростанція 1,1 МВт (введена) + 2,8 МВт (у будівництві).
- Газопоршнева установка (генератор).
- Системи знепилення та нульовий котел (екологічні вимоги JYSK).

**Що шукатимуть для регіонального хабу (гіпотеза):**
- Площа 800–2 000 м² під склад готової продукції та шоурум корпусних меблів.
- Грузовий доступ (ворота під фуру), двір для розвантаження.
- Охорона та відеонагляд.
- Електрика не менш 30–50 кВт (демонстраційне освітлення, дрібне складання).

## 6. «Гачки» — що зачепить саме Accord Import

### Гачок 1 — Регіональний шоурум-склад для дистрибуції на Півдні

Accord Import поставляє меблі в 30+ країн, але внутрішній ринок покриває переважно через Хмельницький (виробництво). Одеська область — центр трьох південних регіонів (Одеська, Миколаївська, Херсонська), які разом є значним ринком для корпусних меблів. Об'єкт 1500 м² з фасадом 28 м + двором 18 соток — готовий профіль для брендованого шоурум-складу.

**Як формулювати:** «1500 м² + двір 18 соток на головній виїзній артерії Одеси — ваша точка входу для дистрибуції на Півдні: Одеська, Миколаївська, Херсонська область. Фасад 28 м = брендування рівня флагману, без витрат на будівництво.»

### Гачок 2 — Логістика експорту через Одеський порт

Accord Import доставляє 96% продукції до ЄС автопарком 140+ вантажівок. З відновленням морського трафіку через Одеський порт маршрут Хмельницький → Одеса → порт → Польща/Данія (JYSK-склади) стає конкурентним до суходільного.

**Як формулювати:** «Якщо JYSK €80M-контракт вимагає нарощення обсягів — Одеський порт скоротить транспортні витрати на ЄС-маршруті. Приміщення на Дальницькій — 12 км від порту; готова база для консолідації/буфферного складу перед відвантаженням.»

### Гачок 3 — Об'єкт заміщує витрати на купівлю

Accord Import будує власну нерухомість (завод $14M). Але для регіонального шоурум-складу купівля нерахобійна: довгострокова оренда 1500 м² за $3000–3500/міс = $36–42K/рік проти $800K–1,2M capex на купівлю аналогічного об'єкту.

**Як формулювати:** «Не потрібно будувати — 1500 м² готового фасадного об'єкту з ремонтом, охороною та 50 кВт з першого дня. Оренда $3000/міс vs купівля від $1M.»

### Гачок 4 — Соседство з JYSK

JYSK відкрив 8-й магазин в Одесі (ТЦ Маяк А1, жовтень 2025) та планує перевершити Данію у 2026. Accord Import як ексклюзивний постачальник JYSK логічно розглядає локації там, де JYSK активно розширюється.

**Як формулювати:** «JYSK відкрив черговий магазин в Одесі. Ви — їхній топ-постачальник. Чому ваші меблі не можна побачити поруч з JYSK-брендом, але від вас — у брендованому шоурумі?»
