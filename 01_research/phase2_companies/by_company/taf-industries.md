# TAF Industries (TAF Drones) — Профіль компанії

> **Короткий висновок (5 рядків):** TAF Industries — найбільший одеський виробник FPV-дронів, $500M виторгу 2025, ~1000 співробітників, 20+ виробничих майданчиків по Україні. Засновник-одесит Олександр Яковенко публічно заявляв, що знайти приміщення для defense-tech практично неможливо — орендодавці відмовляють через страх удару. Компанія змушена купувати нерухомість замість оренди. Об'єкт на Дальницькій закриває їхній профіль точно: промзона, огороджена охоронювана територія, відеонагляд, 50 кВт, зворотний відступ для прийому вантажівок. Стратегія децентралізації (30 приміщень по 40 осіб) означає, що 1500 м² — ідеальний розмір для одного з вузлових майданчиків або для консолідованого одеського хабу.

## 4. Формат нерухомості

### Що відомо про їхні вимоги до приміщень (підтверджено публічно)

**Критична проблема:** Яковенко в публічних інтерв'ю 2024–2025 рр. підтвердив, що пошук приміщень для defense-tech — це «щось зі сфери фантастики». Орендодавці масово відмовляють через страх ракетного удару. Цитата з інтерв'ю Суспільному Одеса:

> «Нам потрібен склад для перетарки наших запчастин і нам всі відмовляють.»  
> «Мені приходиться купляти приміщення, це досить великі витрати.»  
> «Власники пояснюють це ризиком удару.»

**Модель розміщення (із публічних джерел):**
- Децентралізована: 30 приміщень по 40 осіб; уникають великих єдиних об'єктів
- Кожні 3 місяці змінюють локації (ротація для безпеки)
- Перевага — **підвал** (прямо зазначено у Scroll.media 09.2025): «Якщо нам вдається знайти приміщення, яке має підвал, це для нас плюс»
- Потребують промислову зону, не жилу (ризик «сусідів, що помічають»)

**Типові вимоги (виведені з профілю виробництва):**
- Площа на один вузол: 500–1500 м² (40 осіб + складальні столи + склад компонентів + тест-зона)
- Електрика: 50+ кВт (SMT-лінія в Мукачеве споживала промислові потужності)
- Охорона + відеонагляд + огорожа — обов'язково (режимний об'єкт)
- Окремий вхід/в'їзд для вантажних автомобілів (постачання компонентів з Китаю)
- Контроль доступу, без вільного доступу для сторонніх

**Принципове ОБМЕЖЕНня об'єкту:** підвалу немає (це бажане, але не критичне; промзона + огороджена закрита ділянка — фактично аналогічний рівень захищеності та анонімності).

## 6. «Гачки» — чому цей об'єкт цеплятиме TAF конкретно

### Гачок 1. Головний біль компанії — знайти орендодавця, що не злякається
TAF Industries публічно визнає, що пошук нерухомості під defense-tech — «щось зі сфери фантастики», орендодавці відмовляють масово. Яковенко змушений **купувати** приміщення замість оренди. Ми — комерційний орендодавець, який не бояться. Промислово-комерційна зона Хаджибейського району + огороджена ділянка + охоронна система + відеонагляд = **той профіль, якого TAF шукає і не може знайти**.

### Гачок 2. Одеська коріння + локальна логістика
Яковенко — одесит, компанія народилася в Одесі. Виробництво в рідному місті — не чужа локація, а природна точка повернення. Дальницька — головна транспортна артерія з виїздом на трасу: логістика компонентів (Китай → Одеський порт → Дальницька) скорочується до мінімуму.

### Гачок 3. Розмір = один вузол нової моделі
TAF переходить від «30 приміщень по 40 осіб» до гібридної структури. 1500 м² + 1028 м² відкритої території = **комбінований вузол**: складальний цех 600–800 м² + склад компонентів 300–400 м² + офіс 200–300 м² + зовнішня тест-зона у дворі. Точно вписується в їхній «вузловий» формат.

### Гачок 4. Режим безпеки «з коробки»
- Огорожа по периметру + металеві ворота + закритий двір
- Охоронна сигналізація (Легіон), відеонагляд, пожежна сигналізація — все вже встановлено
- Промзона — мінімум жилих сусідів, що можуть «помітити» виробництво
- Декілька незалежних інтернет-провайдерів — критично для режимного об'єкту з IP-телефонією та системами управління

### Гачок 5. 50 кВт 380В — достатньо для SMT-лінії
Їхній SMT-вузол в Мукачеві, знищений у 2025 р., споживав промислові потужності. 50 кВт / 380В — базова мінімальна вимога для сучасної SMT-лінії збирання плат + тест-стенди + освітлення + кліматизація. При необхідності можливе збільшення потужності (відкрите питання в специфікації об'єкту).

### Гачок 6. Стратегія «купувати vs орендувати» — ROI-аргумент
Яковенко сказав, що купує нерухомість, бо ніхто не здає. Але купівля — це іммобілізація капіталу, який міг би йти в R&D. Наша ставка ~$2000–2333/міс ($3000–3500 загалом) — це набагато дешевше альтернативи купівлі промислового об'єкту в Одесі ($200–400К). Ми пропонуємо не купівлю, а ОРЕНДУ — з орендодавцем, що свідомо усвідомлює nature бізнесу.
