# B2B-пищевое производство: заготовительные цехи, кондитерское, dark kitchen, цехи для сетей HoReCa

## Обоснование (почему наш объект подходит)

Объект Дальницкая 48 / Бугаевская 60А подходит под B2B-пищевое производство по ряду ключевых параметров. Центральный водопровод + септик + 380В/50 кВт создают достаточную инженерную базу для пищевого цеха начального-среднего масштаба. Возможна установка приточно-вытяжной вентиляции с выводом на улицу (раздел 8.2 спецификации) — это критическое требование для пекарных, кондитерских и заготовительных производств. Двор 1028 м² позволяет принимать поставки сырья и отгружать готовую продукцию фурами до 10 м без выезда на улицу; ворота 5 м достаточны для рефрижераторов. Промышленно-коммерческая зона Хаджибейского района — типичная среда для пищевых производств: нет ограничений по запаху, шуму производственного цикла, ночной доставке сырья. Отсутствие пешеходного трафика для B2B-производства является нулевым фактором — клиентов-физлиц нет вообще, приезжают только водители и сотрудники.

**Принципиальное ограничение формата:** рассматриваем только цехи без торгового зала для конечного потребителя. Фронтовые кафе, пекарни с витриной «для прохожих» не подходят — промзона без пешего трафика делает такой формат нежизнеспособным. Профильные форматы: dark kitchen (производство для агрегаторов доставки), централизованный заготовительный цех сети HoReCa, кондитерское/хлебобулочное производство B2B для ритейла и HoReCa.

## Summary: топ-3 кандидата и почему именно для нашего объекта

### 1. Региональная пекарня / заготовительный цех B2B (сегментный кандидат, не один бренд)

**Модель: central kitchen (заготовительная фабрика) — производит выпечку/напівфабрикати и поставляет в сети HoReCa, ритейл, кейтеринг.**

Это наиболее реалистичный и горячий сегмент для нашего объекта. В Одесской области — 255 пекарен (poidata.io, 2026), часть из которых уже переросла первоначальные помещения или ищет отдельную производственную площадку для B2B-направления. Объект закрывает ключевые требования: водопровод, канализация, 380В/50кВт, возможность монтажа вытяжки, двор для рефрижераторов. Стандартный пекарный цех в Украине занимает 200–800 м², что позволяет разместить производство с запасом на складирование и офис в пределах 1500 м². Рекомендуемый подход — outreach не к одному бренду, а к 5–10 средним одесским B2B-пекарням и производителям кондитерских изделий, работающим с METRO, Обжорой, Сільпо, кейтерингом. Целевая ставка $2/м² (3000 $/мес за весь объект) — рыночная для производственного помещения в Одессе.

**Ограничение локации:** для этого формата промзона является плюсом, а не минусом. Приоритет — высокий.

### 2. BFI (Bakery Food Investment)

**Модель: B2B-завод (промышленное производство замороженной выпечки для ритейла и HoReCa) — не подходит для нашего объекта напрямую.**

BFI — самый активный инвестор в сегменте (400+ млн грн в новый завод Хуст, 160 млн грн в Козелець, 2025). Однако их формат — крупное автоматизированное производство с 4+ линиями и 3000 т/мес. Наш объект (1500 м², потолки 1,7–3,0 м, 50 кВт) не соответствует масштабу BFI. Как кандидат на аренду — нерелевантен. Тем не менее BFI является индикатором: сегмент замороженной выпечки для B2B в Украине 2025–2026 находится в активной инвестиционной фазе, что создаёт попутный спрос со стороны более мелких аналогов BFI, которые ищут производственные площадки под 500–1500 м².

### 3. MySmak (корпоративное питание + производственная кухня)

**Модель: cloud kitchen / central kitchen — производственная кухня, которая отгружает готовые обеды корпоративным клиентам (офисы, заводы, строительные объекты в промзоне).**

MySmak работает по модели франшизы столовых и корпоративного питания. Производственная кухня площадью 300–1000 м² является ядром операции. Хаджибейский район с его промышленными предприятиями и офисными комплексами — потенциальная клиентская база именно для корпоративного питания. Объект позволяет организовать отдельный производственный цех + офис + склад сырья в пределах 1500 м². Ставка $2/м² рыночная. **Ограничение**: промзона без жилой плотности снижает спрос на розничную часть, но корпоративная модель этого не требует. Приоритет — средний, требует прямого outreach к оператору.

## Влияние локации на ниши

Промышленно-коммерческая зона Хаджибейского района является **естественной средой** для B2B-пищевого производства. Отсутствие жилой плотности и пешеходного трафика не создаёт никаких ограничений для производственных форматов — напротив, снижает риски жалоб от жителей на запахи, шум ночных поставок и грузовой транспорт. Единственный формат из этой ниши, для которого локация создаёт проблему — dark kitchen агрегатора доставки (Glovo Cook Room): курьеры на велосипедах/самокатах работают из точек вблизи жилых кварталов, а не промзон. Все остальные форматы (заготовительный цех, кондитерское B2B, производство полуфабрикатов, корпоративное питание) — приоритет высокий.
