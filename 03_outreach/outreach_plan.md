## Що повинен зробити оператор перед відправкою

Для **кожної** папки `02_priority/tier1/<slug>/`:

1. **Перевірити `letter_uk.md`** — особливо hook на актуальність.
   Новини компанії можуть бути свіжіші, ніж профіль (травень 2026):
   якщо знайдете щось важливе — внесіть у hook (з підтвердженням
   джерела).
2. **Знайти контакт одержувача** — взяти з таблиці вище (канал #1)
   або з `profile.md` → «Контактна стратегія» (канали #2/#3).
3. **Сформувати вкладення:**
   - **Обов'язково:** `Фасад + склад на Дальницкой 1500кв.м..pdf`
   - **Якщо є категоріальний рендер для їхньої ніші** (defense, BPO,
     self-storage, printing — див. колонку «Доп. рендер») → прикласти
     відповідний JPG з `03_outreach/missing_renders/`.
   - **Опційно для top-3 hero** (TAF, Helpware, MasterZoo) →
     персональний рендер з логотипом, якщо оператор зробив окремо
     через `image-prompt-builder`.
4. **Відправити лист + вкладення на email** з verified-адреси у
   таблиці вище. LinkedIn / FB / IG / Telegram / телефон **не
   використовуються** як канал відправки.
5. **Зафіксувати у `03_outreach/sent_log.csv`:**
   ```
   <YYYY-MM-DD>;<компанія>;<канал>;<контакт>;відправлено
   ```
