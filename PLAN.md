# План проекта: поиск арендатора для объекта Бугаевская 60А / Дальницкая 48

## Цель
Найти якорного арендатора для коммерческого объекта 1500 м² в Одессе.
Целевая ставка: 3000–3500 USD/мес. Срок: 30–60 дней.

## Принцип работы
Проект разбит на 6 фаз. Каждая фаза = отдельная сессия Claude Code 
(чтобы не упереться в лимит контекста). Между фазами обмен данными — 
через файлы на диске.

Текущая фаза: см. @CURRENT_PHASE.md
Журнал прогресса: см. @LOG.md

## Фазы

### Фаза 0 — Подготовка (✅ сделано)
- Установка Claude Code
- Создание структуры папок
- @00_object/object_specification.md — спецификация объекта
- @CLAUDE.md — конфигурация проекта
- @PLAN.md, @LOG.md, @CURRENT_PHASE.md — система трекинга

### Фаза 1 — Скелет (в работе)
- @00_object/market_research.md — самостоятельное исследование рынка 
  (генерится Opus-ом)
- Создание 4 субагентов в .claude/agents/:
  niche-researcher, company-researcher, letter-writer, 
  image-prompt-builder
- Создание slash-команд в .claude/commands/:
  research-niches, research-companies, prioritize, generate-outreach

### Фаза 2 — Ресерч сегментов рынка
- Opus составляет финальный список сегментов (10–15 шт)
- Параллельный запуск субагентов niche-researcher по каждому сегменту
- Сводный файл @01_research/phase1_niches/niches_master.md
- Шорт-лист ~50 компаний

### Фаза 3 — Глубокий ресерч компаний
- Параллельные субагенты company-researcher для каждой компании
- Профили в @01_research/phase2_companies/by_company/
- Сводный @01_research/phase2_companies/companies_master.md

### Фаза 4 — Приоритизация
- Деление на Tier 1 / Tier 2 / Tier 3
- Структура папок @02_priority/tier1/<slug>/...
- @02_priority/priority_matrix.md — карта «кому-когда-как»

### Фаза 5 — Outreach assets
- Для каждой компании Tier 1: персональное письмо (UA) + 
  два промпта для генерации картинок
- Артефакты в @02_priority/tier1/<slug>/

### Фаза 6 — Контроль качества и отправка
- Лог отправки @03_outreach/sent_log.csv
- Follow-up через 7 дней по неотвеченным
- Корректировка приоритетов по факту откликов

## Ключевые особенности объекта (для всех агентов)
- Юр. статус: «нежитлові будівлі магазину»
- 1500 м² на 18,2 сотки, фасад 28 м, 380В/50кВт
- История эксплуатации: розничный шоу-рум → офис+производство 
  (Душка). Объект подтверждённо универсален.
- НЕ только ритейл. Рассматриваем розницу, услуги, производство, 
  офисы, логистику, специализированные B2B-форматы.

## Стиль работы
- Отчёты — на русском, Markdown.
- Письма арендаторам — на украинском.
- Источники — всегда с URL.
- Если данных нет — «Дані не знайдено», без выдумок.
- После каждого значимого шага — запись в @LOG.md.
